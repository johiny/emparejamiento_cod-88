# Plantilla: Juego de Emparejar — Robots Cyberpunk (Cables Edition)

Descripción breve:
- Juego de unir conceptos con descripciones usando **cables de neón**.
- Temática: Mantenimiento de robots y sistemas cyberpunk.

Archivos creados:
- [index.html](index.html)
- [style.css](style.css)
- [script.js](script.js) (Lógica de cables SVG y Drag & Drop)
- [data.json](data.json)

Cómo jugar:
1. Haz clic y mantén presionado sobre un **puerto circular** (círculo pequeño) en la columna de la izquierda.
2. Arrastra el cable hasta el puerto del concepto correcto en la derecha.
3. El cable se volverá **Cyan** si es correcto y permanecerá conectado. Si es incorrecto, el cable desaparecerá.
4. Conecta todos los sistemas para ganar.

Personalización:
- **Datos**: Edita [data.json](data.json).
- **Estética**: Cambia los colores neón en las variables `--neon-cyan` y `--neon-pink` en [style.css](style.css).
